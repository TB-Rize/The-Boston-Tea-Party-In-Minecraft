# The-Boston-Tea-Party-In-Minecraft
ボストン茶会事件をテーマにしたマイクラ人狼の亜種です

マインクラフトバージョンは1.21.5から1.21.6までの対応を保証します

もしかすると1.21.7以降でも動作する可能性がありますが未検証です

ワールド単位のミニゲームとなるため既存のサバイバルワールドなどでこれを行うとアイテムロスㇳなどが発生するためミニゲーム専用ワールドを作成すること推奨します

また、参加者にはop権限が与えられる必要があります　opがない場合一部動作が実行できません

ゲーム管理者は以下のコマンドを実行して選定変更権限を入手してください

``` mcfunction
/function tea_party:setting/op
```

このコマンドを実行すると
``` mcfunction
/function tea_party:admin/
```
を実行するように促されるため Admin→ をクリックするかこのコマンドを実行するかのどちらかを行い管理画面を開いてください

# ルール説明
舞台は1773年のボストンでの大事件、ボストン茶会事件です

あなたたちはアメリカを支配するイギリスからの圧政に反抗する組織、サンズオブリバティ(自由の息子達)の一員です

サンズオブリバティは港にやってくるダートマス号、エレノア号、ビーバー号の三隻に積まれている茶葉の束を海に落としてイギリスに対する反抗をアピールすることが目標となります

日中は港付近のタスクを行ってお金を集め、道具を集めましょう

夜になると船への襲撃が可能になります、一定時間内に茶葉を指定量落とし、船に甚大な被害を与えましょう

3隻全ての茶葉を海に落とすことができれば勝利です

## 工作員
もちろんこれはゲームですから、敵がいないとなにもおもしろくありません

工作員、正式名称は「イギリス議会所属工作員」

事態を察知したイギリス議会は既にこちらに伏兵を回しています

工作員の目的は隠密に行動し、船を守り抜くこと

どれか1隻でも茶葉を守り抜くことができれば工作員の勝利です

工作員はクールタイム付きの一撃必殺の武器「暗殺刀」を夜になると受け取り、刀を使用すると周囲に凄まじい殺気を放ちます

殺気を受けたプレイヤーはあまりの恐怖に体が固まり、何も見えなくなります

同時に、抜刀した工作員自身には透明化と高速移動バフがかけられ、殺害してからすぐにその場から逃げることが可能です


# Admin
アドミンにはその場で行える設定のSettingsとタスク、茶葉のスポーン位置などを取得できるItems、ゲーム管理を行うGameの3項目が選択できます
文字をクリックすることで選択した処理を行うことができます

## Settings
役職配分、捨てるべき茶葉の最低量を設定する項目やタスクで与えられる報酬の量、ゲームの制限時間などゲームの根幹となる設定を行うことができます

## Items
実際の茶葉のスポーン位置や廃棄位置、タスクのスポーン位置などは全て管理者が手で置く必要があります

茶葉やタスクのスポーン位置/廃棄位置のアーマースタンドの入手をここで行えます

### Traders / LootTables
タスクで得たお金を使うための商人です

茶葉を捨てるための道具は商人からしか入手できないので商人の設定を行わないと詰みます

商人のスポーン位置と商人の取引内容の設定を行う樽を設置することができます

LootTablesからこのミニゲーム専用アイテムを取得し、商人の取引を設定してください

このデータパックの動作に影響を与えないのであればあらゆるアイテムを取引内容に含めることができます

検証はしていませんがおそらくmod製のアイテムにも対応しています

### 取引内容の樽の諸注意
1.樽は一つしか置けません

2.樽は設置時、保存されている既存の取引内容が中にコピーされ、スムーズに設定できます

3.樽を破壊するときは樽そのものを破壊するのではなく、樽に生えている大きなアーマースタンドを殴って破壊してください

アーマースタンドを殴ることで取引内容が変更されます  樽を直接破壊すると保存されません

4.ゲーム開始中は樽にカギがかけられるのでゲーム中に中のアイテムを取ることはできません

# 特殊アイテム紹介
## マスケット銃
LootTableから取得できる武器の一つです

12秒のリロード時間を持つ高火力の銃で一発18ダメージ(♡9個分)のダメージを与えることができます

準備時間に銃を発射することはできません
